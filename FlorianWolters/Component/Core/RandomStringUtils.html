static public string random(integer $count, integer $start, integer $end, boolean $letters = true, boolean $numbers = true, array $chars = array())

Creates a random string based on a variety of options.

If start and end are both 0, $start is set to ' ' and $end is set to 'z', the ASCII printable characters, will be used, unless letters and numbers are both false, in which case, all printable characters are used that are not alphanumeric.

If $chars is not an empty array, characters will be chosen from the set of characters specified.

Parameters

integer $count The length of the random string to create.
integer $start The position in set of characters to start at.
integer $end The position in set of characters to end before.
boolean $letters true if only letters are allowed.
boolean $numbers true if only numbers are allowed.
array $chars The set of characters to choose randoms from.

Return Value

string The random string.

Exceptions

InvalidArgumentException If $count is less than 0.
